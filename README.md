# 翔啓 / Shokei — AI Developer Portfolio

AIと組んで、仕事を「自動で動く仕組み」に変えるクリエイターのポートフォリオサイト。
機械学習・業務自動化・Claude Code を使った制作実績をまとめています。

🔗 **公開URL:** https://ohkubo3creator-max.github.io/portfolio/

## 技術スタック
- HTML / CSS / JavaScript（フレームワークなし・ビルド不要）
- ダークテーマ（配色はアバターイラストから採色）
- レスポンシブ対応、スクロール演出、作品のライトボックス表示

## ファイル構成
```
.
├─ index.html      # サイト本体（1ページ構成）
├─ css/style.css   # スタイル
├─ js/main.js      # スクロール演出・ナビ・ライトボックス
└─ images/         # 画像（avatar など）
```

## ローカルで開く
`index.html` をブラウザで開くだけで表示できます。
